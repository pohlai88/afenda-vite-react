# Storage Governance

This ruleset governs non-runtime storage roots such as `.legacy/` and `archives/`.

Artifacts in this directory:

- [`storage-governance.md`](./storage-governance.md): policy text
- [`storage-governance.config.json`](./storage-governance.config.json): machine-readable manifest
- [`storage-screening.md`](./storage-screening.md): generated status report

Use this ruleset to keep archived material documented, declared, and separate from active workspace topology.
