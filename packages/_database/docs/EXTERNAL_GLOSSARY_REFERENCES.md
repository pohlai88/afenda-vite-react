# External references — business metadata and glossaries

Use these sources for **governance vocabulary**, **data catalog UX patterns**, and **stewardship** practices. Afenda’s canonical technical bridge remains [`data/business-technical-glossary.yaml`](./data/business-technical-glossary.yaml) and [`DATABASE_ARCHITECTURE_DOCTRINE.md`](./DATABASE_ARCHITECTURE_DOCTRINE.md).

## Standards (conceptual)

| Resource | URL | Notes |
| -------- | --- | ----- |
| DAMA International | [https://dama.org/](https://dama.org/) | Data management body of knowledge; metadata and governance chapters. |
| DAMA Dictionary of Data Management | [Learning resources](https://dama.org/learning-resources/dama-dictionary-of-data-management/) | Shared terminology for teams. |
| ISO/IEC 11179 (metadata registries) | Search ISO catalog | Formal registration of data elements; optional for mature registries. |
